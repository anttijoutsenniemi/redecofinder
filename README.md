# furnival
Tekoälyavusteinen assistant -sovellus tapahtumapakettien tms ehdottamiseen

Miro speksit:

1. Tekoäly esittää kysymyksiä (tarpeiden kartoitus)
Millaisia kalusteita ja koristeita etsit?
Millainen tilaisuus?
Millainen juhlatila?

2. Inspiroidu teemoista
Tekoäly ehdottaa asiakkaalle sopivia juhlateemoja

3. Kerro suosikkisi
Mikä teemavaihtoehto on suosikkisi?

4. Pyydä yhteydenottoa
Lähetä yhteydenottopyyntö. Asiakaspalvelija ottaa sinuun yhteyttä.
